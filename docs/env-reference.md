# duoduo Env Reference

This document is a manual reference for `duoduo-manager`. The app only edits a small basic subset of env keys. If you need more, edit `~/.config/duoduo/.env` directly or extend the app later.

Version baseline:
- `@openduo/duoduo@0.5.0-rc.1`
- `@openduo/channel-feishu@0.5.0-rc.1`

## Managed By duoduo-manager

Daemon:
- `ALADUO_WORK_DIR`
- `ALADUO_DAEMON_HOST`
- `ALADUO_PORT`
- `ALADUO_LOG_LEVEL`
- `ALADUO_PERMISSION_MODE`

Feishu:
- `FEISHU_APP_ID`
- `FEISHU_APP_SECRET`
- `FEISHU_DOMAIN`
- `FEISHU_DM_POLICY`
- `FEISHU_GROUP_POLICY`
- `FEISHU_REQUIRE_MENTION`
- `FEISHU_ALLOW_FROM`
- `FEISHU_ALLOW_GROUPS`

## Notes

- `duoduo-manager` now preserves any env keys outside the basic managed subset.
- Saving from the app will only update the keys listed in `Managed By duoduo-manager`.
- If you want to add more fields back into the UI later, this file is the reference list to use.
